Function vs Arrow function

function ทั่วไป

function calculate_a(a, b){ let sumation = a + b; return (sumation); } document.getElementById("function_nor").innerHTML = calculate_a(20, 30);


Arrow function

const arrow_function = (a, b) => a+b; let reult = arrow_function(20, 30) document.getElementById("function_arrow").innerHTML = reult;


Object with Arrow function

const object_with_arrow = name => ({name:name, city:"CNX"}); let result = object_with_arrow("Earth"); document.getElementById("object_function_arrow").innerHTML = result;
